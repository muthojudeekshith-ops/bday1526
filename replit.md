# Hello Love

## Overview
A static web application featuring an interactive love-themed experience with animated hearts, password-protected entry, and an audio conversation playback feature.

## Project Architecture
- **Type**: Static website (HTML, CSS, JavaScript)
- **Directory**: All content served from the `docs/` folder
- **Assets**: Audio files in `docs/audio/z1/` and `docs/audio/z2/`

## How It Works
1. Landing screen with animated hearts and "HELLO LOVE" title
2. Password entry screen (password: `c5d9`)
3. Earphones warning screen
4. Audio conversation playback with typed text synchronized to audio clips

## Running Locally
Served via Python's built-in HTTP server on port 5000 from the `docs` directory.

## Deployment
Configured as a static site deployment serving the `docs` directory.
